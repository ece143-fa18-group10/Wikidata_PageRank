# California_PageRank

## Dataset
- Source: http://www.cs.cornell.edu/courses/cs685/2002fa/data/gr0.California
- Size: 25814 lines
- Dataset sample:
  - `n 0 http://www.berkeley.edu/` (the page index of http://www.berkeley.edu/ is 0)
  - `e 80 3301` ( there is a link from page 80 to page 3301)
